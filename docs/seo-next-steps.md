# Orvyn SEO Next Steps

## Google Search Console

1. Open Google Search Console.
2. Add `https://orvyn.cc` as a Domain property if DNS verification is available, or as a URL-prefix property if you want the fastest setup.
3. Complete ownership verification using the method Google provides.
4. Submit the sitemap:
   `https://orvyn.cc/sitemap.xml`
5. Use URL Inspection for these pages after deployment:
   - `https://orvyn.cc/`
   - `https://orvyn.cc/services`
   - `https://orvyn.cc/about`
   - `https://orvyn.cc/case-studies`
   - `https://orvyn.cc/contact`
6. Request indexing for the homepage first, then the main inner pages.

## Bing Webmaster Tools

1. Open Bing Webmaster Tools.
2. Import the verified property from Google Search Console, or add `https://orvyn.cc` manually.
3. Submit the same sitemap:
   `https://orvyn.cc/sitemap.xml`
